# Envio_Cenario

Projeto para enviar um cenário por e-mail dos dados atualizado.

Projeto criado em Python, HTML e CSS.

Projeto inclui:

- Conexão com bd
- Conexão com app de envio de e-mail
- Pesquisas SQL
